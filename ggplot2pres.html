The grammar

  • data = your data that you want to visualize
  • aesthetic, aes = a mapping
  • geom = a shape that represents your data
  • stat = a statistical transformation

an example plot

giris <- ggplot(iris, aes(x = Sepal.Width,
                          y = Petal.Width, group = Species)) + 
  geom_point(shape = 19,
             aes(color = Species), position = "jitter") +
  geom_smooth(method = "lm", aes(color = Species)) + 
  labs(title = "Three species of Iris
       comparison of petal width and sepal width",
       x = "Sepal Width",
       y = "Petal Width")

setting defaults

you can set defaults so you don't have to specify them every time you plot

test.df <- data.frame(a = 1:10, b = 1:10)

plot with package defaults

ggplot(test.df, aes(x =a, y = b)) + geom_point() +
  labs(title ="Some test data")

setting theme defaults

new.theme <- theme_bw() + theme(text = element_text(size = 16))
theme_set(new.theme)
update_geom_defaults("point", list(shape = 19))

plot again, with new defaults

ggplot(test.df, aes(x =a, y = b)) + geom_point() +
  labs(title = "Some test data")
